************
Installation
************

Install with `pip`:
-------------------

.. code-block:: sh

   pip3 install dpay-cli

Manual installation:
--------------------

.. code-block:: sh

   git clone https://github.com/dpays/dpay-cli
   cd dpay-cli
   python3 setup.py install --user

Upgrade
-------

.. code-block:: sh

   pip install --user --upgrade dpay-cli
