************
Installation
************

Install with `pip`:
-------------------

.. code-block:: sh

   pip3 install creacli

Manual installation:
--------------------

.. code-block:: sh

   git clone https://github.com/creativechain/creacli
   cd creacli
   python3 setup.py install --user

Upgrade
-------

.. code-block:: sh

   pip install --user --upgrade creacli
