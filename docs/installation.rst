************
Installation
************

Install with `pip`:
-------------------

.. code-block:: sh

   pip3 install dpaycli

Manual installation:
--------------------

.. code-block:: sh

   git clone https://github.com/dpays/dpaycli
   cd dpaycli
   python3 setup.py install --user

Upgrade
-------

.. code-block:: sh

   pip install --user --upgrade dpaycli
